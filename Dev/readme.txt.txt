Installation:
1) Download the files
2) Unzip "Skuld" folder
3) Enter newly unzipped "Skuld" folder
4) Enter "Game" file
5) Enjoy playing the game!



Controls for playing the game:
WASD - Movement
SpaceBar/Enter - Interact with textboxes/NPCs
Escape - Open/close menu
